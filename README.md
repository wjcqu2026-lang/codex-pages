# Codex 网页原型合集

这是一个可直接发布到 GitHub Pages 的静态网站。

## 发布方式

1. 在 GitHub 新建一个仓库，例如 `codex-pages`。
2. 上传本目录下的所有文件和文件夹。
3. 进入仓库的 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub Pages 部署完成。

部署完成后，首页地址通常是：

```text
https://你的用户名.github.io/codex-pages/
```

每个原型页面都可以从首页进入。
